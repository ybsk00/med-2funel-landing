# 병원 설정 간단 입력

> **아래 6가지만 입력하면 됩니다!** (나머지는 과목별 자동 적용)

---

## 필수 입력

```
[1] 병원명: 
[2] 원장명: 
[3] 과목: (피부과/치과/정형외과/비뇨기과/내과/암요양병원/한의원/성형외과/소아과/신경외과/산부인과 중 선택)
[4] 주소: 
[5] 전화번호: 
[6] 사업자등록번호: 
```

## 선택 입력 (비워두면 자동 생성)

```
[7] 헬스케어 로고: (예: OO헬스케어 - 병원명 노출 금지)
[8] 메디컬 로고: (예: OO피부과 - 병원명 노출)
[9] 히어로 문구: 
[10] AI 이름: (예: 에밀리, 소피 등)
```

---

## 예시

```
[1] 병원명: 글로우피부과
[2] 원장명: 이수정
[3] 과목: 피부과
[4] 주소: 서울 서초구 강남대로 123
[5] 전화번호: 02-555-1234
[6] 사업자등록번호: 111-22-33333
[7] 헬스케어 로고: 글로우헬스케어
[8] 메디컬 로고: 글로우피부과
[9] 히어로 문구: 베이스가 달라지는 광채 루틴 리셋
[10] AI 이름: 에밀리
```

**입력 완료! 10분 후 사이트 오픈 가능합니다.**
