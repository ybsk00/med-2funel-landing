# 1퍼널 채팅형 헬스케어 스크립트 (의료법 준수 · 재미/흥미 · 리드발굴 · 회원가입 전환)

> 목적: **로그인 전(1퍼널)** 단계에서 사용자가 **재미있게 참여**하도록 만들고, **잠재고객(리드) 발굴** 및 **회원가입(로그인) 전환**을 유도한다.  
> 원칙: 본 스크립트는 **진단/처방/치료 보장**을 하지 않으며, **생활 습관·웰니스 점검(참고용)** 범위에서만 안내한다.

---

## 0. 안전 문구(의료법 리스크 최소화) — 반드시 전 구간에 적용

### 0-1) 시스템 고정 안내(상단 고정 또는 첫 메시지)
- “이 채팅은 **진단이나 처방이 아닌 생활 습관·웰니스 점검(참고용)** 입니다.”
- “정확한 상태 판단과 치료 여부는 **의료진 상담을 통해 확인**이 필요합니다.”
- “응급 증상(흉통, 호흡곤란, 의식저하 등)이 있으면 즉시 응급의료기관을 이용하세요.”

### 0-2) 금지 표현(운영 가이드)
- 금지: “당신은 ○○입니다(확정)”, “○○ 치료가 필요합니다(단정)”, “완치/효과 보장”, “약/한약/시술 처방 제시”
- 허용(권장): “가능성/경향/신호”, “참고용”, “생활 관리 팁”, “상담 시 참고 자료 정리”

---

## 1. 1퍼널 전환 구조(재미 + 리드발굴 + 가입 유도)

### 1-1) 참여를 재미있게 만드는 UX 장치(권장)
1) **퀘스트/미션 톤**: “3분 건강퀘스트 시작”  
2) **진행률/레벨업 피드백**: “2/7 완료 · 집중력 +1”  
3) **버튼형 선택(2~4개)**: 긴 서술 입력 최소화  
4) **즉시 보상(팁 1개)**: 나머지 혜택은 로그인 후 제공  
5) **티저(결과 일부만 공개)**: “현재 패턴은 A 쪽(상세는 로그인 후)”  
6) **저장/추세(회원가입 혜택)**: “결과 저장 · 주간 추세 리포트”

### 1-2) 2단 게이팅(리드 수집 단계)
- **A단계(라이트 리드)**: 4~5문항 후 → “결과 저장/상세 리포트”를 **간편가입(휴대폰/이메일/카카오)**로 제공
- **B단계(딥 리드)**: 로그인 후 리포트 직전 → “상담에 도움이 되도록 3문항 추가 정리”

## 2. 공통 채팅 플로우 템플릿 (모든 모듈 공통)

### 2-1) 오프닝(공통)
**봇:**  
“3분 건강퀘스트를 시작할게요. 이 대화는 **진단이 아닌 생활 리듬 점검(참고용)** 입니다.”  
“원하시면 결과를 저장하고, 나중에 상담 시 참고할 수 있도록 정리해 드릴 수 있어요.”  

**버튼:**  
- [시작하기]  
- [어떤 내용인가요?]

**(사용자가 ‘어떤 내용인가요?’ 선택 시)**  
**봇:**  
“7개의 짧은 질문으로 생활 습관 패턴을 점검하고, **요약 결과(참고용)** 와 **생활 팁**을 드립니다.  
상세 리포트·추세 저장은 로그인 후 제공돼요.”  
**버튼:** [시작하기]

### 2-2) 질문 루프(총 7문항 권장)
- 문항 구성: `질문 1개 + 선택지 2~4개`
- 중간 피드백(2문항마다 1회):
  - “진행률 30% 완료”
  - “오늘의 팁 1개 지급(짧게)”

### 2-3) 중간 게이팅(4문항 완료 시점)
**봇:**  
“여기까지 답변으로 **패턴이 거의 잡혔어요.**  
결과를 저장하고 **상세 리포트(맞춤 루틴/체크리스트 포함)** 를 보려면 로그인/간편가입이 필요해요.”

**버튼:**  
- [간편가입하고 상세 보기]  
- [일단 요약만 보기]

### 2-4) 로그인 전 요약(‘요약만 보기’ 선택 시)
**봇:**  
“요약(참고용): **A 신호 ↑ / B 신호 →**  
생활 팁: 1) ___ 2) ___  
더 자세한 분석과 기록 저장은 로그인 후 가능합니다.”  
**버튼:** [간편가입] [종료]

### 2-5) 로그인 후(상세 리포트 + 딥 리드 3문항)
**봇:**  
“상세 리포트가 준비됐어요.  
원하시면 의료진 상담 시 도움이 되도록 **3문항만 더 정리**할까요?”

**버튼:**  
- [3문항 추가 정리]  
- [리포트만 보기]  
- [상담 연결(선택)]

**(3문항 예시 — 공통 딥리드)**  
1) “언제부터 불편했나요?” (오늘/1주 이내/1달 이내/3달+)  
2) “일상 영향은?” (거의 없음/가끔 불편/자주 불편/매우 불편)  
3) “원하는 도움은?” (생활관리/검사·상담 문의/예약 희망/기타)

## 4. 로그인(회원가입) 유도 카피 3종(선택 적용)

1) “결과를 저장하고 **상세 리포트(맞춤 루틴/체크리스트)** 를 보려면 간편가입이 필요해요.”  
2) “지금까지 답변으로 **패턴 분석이 거의 완료**됐어요. 로그인하면 ‘내 리듬 리포트’를 바로 드릴게요.”  
3) “다음부터는 **이어하기/추세 보기**가 가능해요. 간편가입 후 기록을 저장해 둘까요?”
