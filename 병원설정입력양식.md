# 병원 AI 헬스케어 플랫폼 - 설정 입력 양식

> 아래 내용을 채워주시면 자동으로 병원 설정이 생성됩니다.
> 모르는 항목은 비워두시면 기본값이 적용됩니다.

---

## 1. 병원 기본 정보

```
병원명: 
원장명: 
원장 직함: 대표원장

주소: 
전화번호: 
팩스: 
사업자등록번호: 

네이버 검색 키워드: (예: 강남 피부과, 역삼 다이어트)
```

---

## 2. 과목 선택

아래 중 하나를 선택하여 적어주세요:

```
과목: 피부과
```

**선택 가능한 과목:**
- 피부과 (dermatology)
- 치과 (dentistry)
- 정형외과 (orthopedics)
- 비뇨기과 (urology)
- 내과 (internal-medicine)
- 암요양병원 (oncology)
- 한의원 (korean-medicine)
- 성형외과 (plastic-surgery)
- 소아과 (pediatrics)
- 신경외과 (neurosurgery)
- 산부인과 (obgyn)

## 3. 헬스케어 영역 (비로그인 - 로그인 유도용)

### 브랜딩
```
헬스케어 로고 텍스트: (예: OO헬스케어, OO뷰티 - 병원명 노출 금지)
```

### AI 페르소나 (헬스케어)
```
AI 이름: (예: 에밀리, 소피, 라이언 등)
톤앤매너: (예: 능글맞고 유머러스함 / 친절하고 따뜻함 / 세련되고 profesional)
```

### 히어로 섹션
```
히어로 헤드라인: (예: 베이스가 달라지는 광채 루틴 리셋)
히어로 서브헤드라인: (예: 지금 내 상태를 빠르게 체크하고...)
```

## 4. 메디컬 영역 (로그인 후 - 예약 유도용)

### 브랜딩
```
메디컬 로고 텍스트: (예: OO피부과 - 병원명 노출)

### AI 페르소나 (메디컬)
```
