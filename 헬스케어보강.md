# 11개 병원(과) 랜딩페이지 세션 보강안 (2중 퍼널 구조)

## 공통 전제: 2중 퍼널 UX
- 비회원(External / SLM): “자가 체크 + 생활관리 코치 + 콘텐츠”로 체류시간 확보 → 3턴 내 로그인 유도
- 회원(Internal / LLM): “의료진 기반 상담 + 예약 + 개인화 플랜”으로 내원 전환
- 법적/리스크 가드(공통 문구):
  - 본 서비스는 일반적인 건강 정보 제공이며, 진단·처방을 대체하지 않습니다.
  - 증상이 지속되거나 악화되면 의료진 상담 및 내원을 권장합니다.
  - 개인 상태에 따라 결과가 다를 수 있습니다.

# 1) 피부과 - 글로우헬스 (/healthcare/dermatology)

## 세션 A. “피부 고민 60초 정리” (비회원 SLM)

- UI: [피부 고민 선택 칩] 여드름/기미/홍조/모공/탄력/민감

## 세션 B. “생활 루틴 처방전” (비회원 SLM)
- 3카드 구성:
  1) 세안·보습 루틴 리셋: 자극 최소화 체크리스트
  2) 자외선/색소 관리: 계절·실내외 상황별 가이드
  3) 트러블 악화 요인 점검: 수면/식습관/마스크 마찰

## 세션 C. “회원 전용: 피부 상태 기반 맞춤 플랜” (회원 LLM)

- 구성:
  - 문진 요약 → 예상 이슈 분류 → 내원 시 검사/진료 포인트 안내
  - 예약 연결: “가까운 진료 가능 시간 확인”
- CTA: “로그인 후 개인 플랜 받기”

## 세션 D. FAQ (의료광고 리스크 최소화)
- Q1. 사진만으로 진단하나요? → “진단은 내원 후 의료진 판단, 앱은 상담 준비를 돕는 도구”
